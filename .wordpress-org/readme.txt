== Description ==

Transform your multilingual WordPress site with AI-powered translation. This free plugin integrates seamlessly with Polylang to provide one-click translation using OpenAI's advanced language models.

= Features =

* **One-Click Translation**: Translate posts and pages with a single click
* **OpenAI Integration**: Leverage GPT-4, GPT-4 Turbo, and GPT-3.5 models
* **Seamless Polylang Integration**: Works directly within your Polylang workflow
* **Multi-Language Support**: Translate to any language supported by Polylang
* **User-Friendly Interface**: Simple and intuitive translation interface

= Requirements =

* WordPress 5.8 or higher
* PHP 8.1 or higher
* [Polylang](https://wordpress.org/plugins/polylang/) plugin installed and activated
* OpenAI API key

== Installation ==

1. Download the latest release from the [GitHub Releases](https://github.com/epicwp/Polylang-AI-Automatic-Translation-Plugin/releases) page
2. Upload the plugin ZIP file through the 'Plugins > Add New > Upload Plugin' menu in WordPress
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to Settings → Polylang AI Translation
5. Enter your OpenAI API key
6. Start translating your content!

= Getting an OpenAI API Key =

1. Sign up for an account at [OpenAI](https://platform.openai.com/)
2. Navigate to the API section
3. Generate a new API key
4. Copy the key and paste it in the plugin settings

== Frequently Asked Questions ==

= Do I need Polylang to use this plugin? =

Yes, this plugin requires Polylang (free or pro version) to be installed and activated.

= Which AI models are supported? =

The free version supports OpenAI's GPT-4, GPT-4 Turbo, and GPT-3.5 models. The premium version includes additional providers like Claude, Gemini, and OpenRouter.

= How much does OpenAI cost? =

OpenAI charges per token (word). Costs vary by model. Check [OpenAI's pricing page](https://openai.com/pricing) for current rates.

= Can I translate custom post types? =

Yes, the plugin works with any post type that Polylang supports.

= Is there a translation limit? =

The free version is limited to single post/page translations. For bulk translation, upgrade to the premium version.
