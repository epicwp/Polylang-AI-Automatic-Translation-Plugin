== Description ==
Adds AI translation via a GPT model capability to Polylang. This plugin enhances Polylang by providing automatic translation functionality using advanced AI technology.

Features:
* Automatic translation using GPT models
* Seamless integration with Polylang
* Support for multiple languages
* Easy to use interface
* Custom translation settings

== Installation ==
1. Upload the plugin files to `/wp-content/plugins/polylang-ai-autotranslate` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure your translation settings under the Polylang menu
