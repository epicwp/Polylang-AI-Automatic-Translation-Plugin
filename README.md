# Polylang AI Automatic Translation (Free)

Free version of Polylang Automatic Translation with AI. Translate your WordPress content effortlessly using OpenAI's powerful GPT models.

## Features

- **One-Click Translation**: Translate posts and pages with a single click
- **OpenAI Integration**: Leverage GPT-4, GPT-4 Turbo, and GPT-3.5 models
- **Seamless Polylang Integration**: Works directly within your Polylang workflow
- **Multi-Language Support**: Translate to any language supported by Polylang
- **User-Friendly Interface**: Simple and intuitive translation interface

## Requirements

- WordPress 5.8 or higher
- PHP 8.1 or higher
- [Polylang](https://wordpress.org/plugins/polylang/) plugin installed and activated
- OpenAI API key

## Installation

1. Download the latest release from the [Releases](https://github.com/epicwp/Polylang-AI-Automatic-Translation-Plugin/releases) page
2. Upload the plugin files to `/wp-content/plugins/polylang-ai-automatic-translation-free` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to Settings → Polylang AI Translation
5. Enter your OpenAI API key
6. Start translating your content!

## Getting an OpenAI API Key

1. Sign up for an account at [OpenAI](https://platform.openai.com/)
2. Navigate to the API section
3. Generate a new API key
4. Copy the key and paste it in the plugin settings

## Usage

### Translating a Single Post or Page

1. Edit the post or page you want to translate
2. Look for the Polylang language switcher in the post editor
3. Click on the language you want to translate to
4. Click the "Translate with AI" button
5. Wait for the translation to complete
6. Review and publish the translated content
